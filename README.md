# Rails Template

### Add Stimulus - simple data binding
`bundle exec rails webpacker:install:stimulus`

### Add Ember - going all in on JS
`bundle exec rails webpacker:install:vue`

### Add Vue.js - if you somehow dont wanna use Ember
`bundle exec rails webpacker:install:vue`
